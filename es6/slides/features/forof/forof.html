For of

what we wanted for in to be
  • Loops over the data unlike for in which loops properties
  • Only works on iterable objects
Example straight from the MDN 

let arr = [3, 5, 7];
arr.foo = "hello";

for (let i in arr) {
   console.log(i); // logs "0", "1", "2", "foo"
}

for (let i of arr) {
   console.log(i); // logs "3", "5", "7"
}
